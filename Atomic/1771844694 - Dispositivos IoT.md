# Dispositivos IoT

**Sistema embebido**: sistema de computación compuesto por hardware y software que está diseñado para realizar una o pocas funciones específicas dentro de un sistema más grande. Está optimizado para una tarea concreta.

Y los dispositivos de IoT también suelen ser sistemas embebidos, por lo que simplemente lo definiremos y le daremos una idea de lo que son los sistemas integrados. 

>[!DANGER]
>La eficiencia de un sistema embebido se refiere a: **eficiencia energética, uso optimizado de recursos, rendimiento en tiempo real, tamaño compacto y reducción de costos**.

**Restricciones en sistemas embebidos**

- **Restricciones económicas y de tiempo:**
	- Costo: se debe equilibrar la funcionalidad deseada con un costo asequible para el producto final
	- Tiempo de desarrollo: los sistemas integrados se desarrollan bajo estrictas limitaciones de tiempo, lo que requiere procesos optimizados y componentes cuidadosamente seleccionados.
- **Restricciones Ambientales y de aplicación**:
	- Entorno operativo: los sistemas deben diseñarse para operar de manera confiable en entornos específicos e impredecibles.
	- Interacción con el mundo físico: requiere un manejo cuidadoso la integración de sensores y actuadores
- **Restricciones de desarrollo y escalabilidad**:
	- Complejidad
	- Seguridad
	- Escalabilidad

Características principales:
- Función dedicada: se enfoca en una tarea específica en lugar de ser multifuncional
- Integrado: el hardware y el software están embebidos en un dispositivo más grande, formando parte de él.
- Diseño específico: se diseña con optimizaciones para cumplir con sus requisitos específicos, como consumo de energía, tamaño, rendimiento...
- Suelen tener un microprocesador, microcontrolador, memoria y otros periféricos para realizar sus tareas.
- Pueden no tener sistema operativo

**Sistema Dispositivo IoT**: es una red de objetos físicos que integran sensores, software y otras tecnologías para conectarse a Internet, permitiéndoles recopilar, intercambiar y actuar sobre los datos recopilados del mundo físico, lo que facilita la automatización, el monitoreo y el control remoto de diversas tareas.

Componentes claves:
- **Sensores**: recolectan datos del entorno
- **Actuadores**: reciben instrucciones para realizar una acción física en el mundo real.
- **Interfaces de red**: permiten la conexión a Internet y la comunicación con otros dispostivos en la nube.
- **Software/Procesamiento**: permite que el dispositivo procese los datos y siga instrucciones.

Casi todos los sistemas embebidos siguen una [[1771846300 - Estructura genérica de un sistema embebido|Estructura genérica de un sistema embebido]].
