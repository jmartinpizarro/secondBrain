# Estructura genérica de un sistema embebido

Basada en:
- ADC (Analog-to-Digital Converter): convierte las señales analógicas de los sensores en datos digitales para su procesamiento
- [[1772447134 - Microcontrolador|Microcontrolador]]: un procesador de propósito general que gestiona el sistema y realiza tareas de control
- [[1772449096 - Sensores y Actuadores|Sensores y Actuadores]]: los sensores capturan la información del entorno; mientras que los actuadores son interactúan con el entorno físico, controlados por las señales analógicas de la salida.
- IP (Intellectual Property): bloques de funciones de software o hardware predefinidas que se integran en el sistema
- FPGA (Field-Programmable Gate Array): circuito integrado reconfigurable utilizado para implementar funciones de procesamiento de señal digital intensivas, proporcionando flexibilidad y rendimiento.
- DAC (Digital-to-Analog Converter): convierte los datos digitales procesados de nuevo en señales analógicas.
