# Introducción a los compiladores

## Definiciones ##

- Ensamblador: compilador sencillo. El lenguaje fuente representa instrucciones en código máquina. Proporciona facilidades de mayor nivel: etiquetas de salto, macros...
- Traductor: traduce un programa de un lenguaje de alto nivel a otro lenguaje de alto nivel.
- Intérprete: traductor o compilador incremental. Analiza, pre-compila y ejecuta simultáneamente. Cada función introducida se almacena pre-compilada. Si se encuentra un error, se modifica la función y se recompila sin modificar el resto.
- Compilador cruzado: compilador que genera código para un ordenador distinto.
- Autocompilador: compilador escrito en el propio lenguaje que compila. Facilita la portabilidad
- Metacompilador: programa que recibe la descripción de un lenguaje y genera un compilador para ese lenguaje
- Descompilador: recibe código máquina y lo traduce a un lenguaje de alto nivel.
