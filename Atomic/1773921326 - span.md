# span

Just a pointer and a size.
- Again, a non-owning view, defined in header `<span>`
- Passed by value
- Lifetime is a precondition
- **It can mutate**, unlike most 'view types'
- *constexpr* ready

>[!DANGER]
>If we are going to travel through a vector, array... it is much better to use a generic interface such as `span` and then use it as a light-weight interface.

```cpp
void print_reverse(span<int> si) {
	for (auto i : std::ranges::reverse_view(si)) {
		cout << i << '';
	}
	cout << "\n";
}

int main (){
	vector<int> v1 = {1,2,3,4,5};
	print_reverse(v1); // 5,4,3,2,1
	
	span<int> si {v1};
	print_reverse(s1.first(2)); // 2,1 - only the first two numbers
	print_reverse(s1.last(2)); // 5,4 - only the last two numbers

	
}
```
