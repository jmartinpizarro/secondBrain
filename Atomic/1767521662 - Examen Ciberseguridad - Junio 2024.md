# Examen Ciberseguridad - Junio 2024

**Problem 1**

Las *capabilities* dividen los privilegios tradicionales Linux del usuario *root* en permisos finos e independientes. Un proceso solo recibe las *capabilities* necesarias para su ejecución.

Busca seguir el principio del mínimo privilegio, en el cual un programa o entidad solo mantiene los privilegios mínimos para poder realizar sus tareas de manera correcta.
