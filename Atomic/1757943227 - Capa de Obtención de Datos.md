# Capa de Obtención de Datos

Modos de obtención:
- Batch: intervalos periódicos. Adecuado cuando no es necesaria la inmediatez
- Streaming: los datos se procesan a medida que llegan. Útil para aplicaciones que requieren inmediatez.
