# Análisis Semántico

Comprueba que el código generado tiene sentido de manera inherente (una función que debe retornar `int` no puede retornar `123.456`).

Las **gramáticas independientes del contexto (G2)** no son suficientes para realizar el análisis semántico.  Para ello se han creado las **gramáticas atribuidas**, donde destaca la **verificación de tipos**.
